## 6. Editor Setup Tool 실행 정책

모든 Tools > Setup 계열 에디터 툴은 **사용자가 명시적으로 실행**하는 것을 기본 정책으로 합니다.

- InitializeOnLoad 또는 EditorApplication.delayCall을 통한 자동 실행을 사용하지 않습니다.
- Setup Tool은 **Idempotent** 하게 동작합니다. 여러 번 실행해도 기존 에셋을 덮어쓰지 않습니다.
- 에셋이 이미 존재하면 스킵하고, 없는 것만 신규 생성합니다.
- **기존 에셋을 수정하거나 덮어쓰지 않습니다.** 기존 에셋을 변경해야 하는 경우에는 별도의 `Tools > Migration` 또는 `Tools > Upgrade` 기능으로 제공합니다.
- 실행 결과는 Console 로그와 Dialog로 명확히 보고합니다.
