## 1. 레벨 시스템

### 1.1 기본 설정
- **시작 레벨**: Lv.1
- **최대 레벨**: 무한 (계속 상승 가능, 버전 업데이트마다 확장)
- **레벨 상한선** (v0.2.0): Lv.20

### 1.2 경험치 획득

**경험치 = 손님 만족도 * 기본 경험치**

| 별점 | 배수 | 기본 EXP | 예시 |
|------|------|---------|------|
| 5별 | 3.0배 | 200EXP | 600EXP |
| 4별 | 2.0배 | 200EXP | 400EXP |
| 3별 | 1.0배 | 200EXP | 200EXP |
| 2별 | 0.5배 | 200EXP | 100EXP |
| 1별 | 0.2배 | 200EXP | 40EXP |

**설계 의도**: 성공 플레이에 큰 보상, 실패해도 진행 가능

### 1.3 레벨별 경험치 요구량

| 레벨 | 필요 EXP | 누적 EXP |
|------|---------|---------|
| Lv.1 | 0 | 0 |
| Lv.2 | 500 | 500 |
| Lv.3 | 700 | 1,200 |
| Lv.4 | 900 | 2,100 |
| Lv.5 | 1,100 | 3,200 |
| Lv.6 | 1,400 | 4,600 |
| Lv.7 | 1,700 | 6,300 |
| Lv.8 | 2,000 | 8,300 |
| Lv.9 | 2,300 | 10,600 |
| Lv.10 | 2,600 | 13,200 |
| Lv.11+ | 3,000+ (계속 증가) | - |

**설계 의도**:
- 초반 빠른 레벨업 (동기 부여)
- 중반 이후 천천히 상승 (장기 플레이)

## 2. 레벨별 해금 시스템

### 2.1 레벨 1-5: 기초 마스터 단계

**Lv.1**
- 시작 - 유자차, 말차 기본 제공
- 기본 재료 해금: 꿀, 우유, 생강

**Lv.2**
- 새 재료 해금: 계피, 레몬

**Lv.3**
- 인테리어 시스템 해금 (기본 벽지)
- 새 재료 해금: 정향

**Lv.4**
- 상점 확장 (더 많은 인테리어 선택)

**Lv.5**
- **새 차 해금: 보이차, 연꽃차** ⭐
- 새 재료 해금: 생크림, 카다몬
- 컬렉션 시스템 기본 해금

### 2.2 레벨 6-10: 중급 도전 단계

**Lv.6**
- 새 인테리어 테마 해금 (국가별 테마)
- 특별 재료 상점 해금

**Lv.7**
- 손님 친숙도 시스템 강화
- 추가 조명 옵션 해금

**Lv.8**
- 스토리 진행 1단계
- 새로운 손님 토너먼트 시스템 (예정)

**Lv.9**
- 손님 특별 요청 시스템 해금

**Lv.10**
- **최종 차 해금: 쉬트 티 차이** ⭐
- 문화카드 시스템 전체 해금
- 추가 테이블 구매 가능

### 2.3 레벨 11-15: 고급 콘텐츠 단계

**Lv.11**
- 스토리 진행 2단계 시작
- 프리미엄 인테리어 해금

**Lv.12**
- 특별 이벤트 시스템 해금

**Lv.13**
- 손님 선물 시스템 해금
- 장식품 세트 해금

**Lv.14**
- 특별 재료 조합 해금

**Lv.15**
- 스토리 진행 3단계 (중반 진행)
- 최고급 인테리어 테마 해금

### 2.4 레벨 16-20: 마스터 단계

**Lv.16-19**
- 특별 스토리 이벤트 추가
- 숨겨진 재료 조합 추가

**Lv.20**
- 최종 스토리 엔딩 준비
- 모든 콘텐츠 언락 가능

## 4. 해금 시뮬레이션

### 이상적인 진행 순서

```
Day 1 (Lv.1)
- 유자차, 말차 제공 시작
- 기본 재료: 꿀, 우유, 생강

Day 2-3 (Lv.2-3)
- 계피, 레몬 추가
- 인테리어 시스템 시작 (1-2가지 벽지)

Day 4-5 (Lv.4-5)
- 보이차, 연꽃차 해금 ⭐
- 컬렉션 시작

Day 6-10 (Lv.6-10)
- 국가별 테마 인테리어
- 쉬트 티 차이 해금 ⭐
- 모든 차 이용 가능

Day 11-20 (Lv.11-15)
- 스토리 진행 2-3단계
- 프리미엄 콘텐츠 추가

Day 21+ (Lv.16+)
- 최종 스토리 진행
- 완전 마스터 상태
```

## 5. 콜렉션 시스템

### 5.1 수집 대상
1. **차 문화카드** (5종)
   - 각 차마다 카드 1장
   - 문화 정보, 이미지, 스토리 포함

2. **국가 정보카드** (5개)
   - 각 국가마다 카드 1장
   - 국가 배경, 전통, 특징

3. **손님 스토리카드** (5명)
   - 각 손님마다 카드 3-5장
   - 친숙도 올라가면서 해금

### 5.2 해금 조건

**차 문화카드**:
- 해당 차를 처음 사용하면 자동 해금
- 예: 유자차 첫 제공 → 유자차 카드 해금

**국가 정보카드**:
- Lv.10 달성 후 자동 해금
- 각 국가별 정보 공개

**손님 스토리카드**:
- 친숙도 올라가면서 점진적 해금
- Lv.10 이후 더 많은 정보 공개
