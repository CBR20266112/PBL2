| 난이도 | ⭐ 쉬움 |

## 7. 재료 조합 팁

### 기본 조합
- 유자차 + 꿀 = 더 달콤함
- 말차 + 우유 = 더 부드러움
- 보이차 + 생강 = 더 따뜯음
- 연꽃차 + 레몬 = 더 신선함
- 차이 + 우유 = 더 순한맛

### 고급 조합
- 말차 + 생크림 + 정향 = 프리미엄
- 보이차 + 꿀 + 생강 = 건강식
- 연꽃차 + 카다몬 + 꿀 = 신비로움
- 차이 + 모든 향신료 = 마스터급
