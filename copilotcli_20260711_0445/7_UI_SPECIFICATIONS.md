# UI 스펙시트 (UI Specifications)

**버전**: v0.2.0  
**마지막 수정**: 2026-07-11  
**목적**: 모든 화면의 레이아웃, 요소, 상태 정의

---

## 1. UI 디자인 원칙

### 1.1 핵심 철학
- **귀여움**: 둥근 모서리, 부드러운 색감
- **따뜻함**: 주황, 베이지, 연초록 색상 계열
- **명확함**: 터치 영역 충분히 큼 (최소 44×44px)
- **접근성**: 큰 글꼴, 명확한 대비

### 1.2 표준 규격
- **스크린**: 1080px × 2340px (주요 Android 해상도)
- **가이드라인**: 8px 그리드 기반
- **폰트**: 기본 16sp, 제목 24sp+
- **터치 타겟**: 최소 44×44dp

## 2. 메인 화면 (Home/Lobby)

### 2.1 구성 요소

```
┌─────────────────────────────────┐
│ 시간 | Lv.5 | 보유금: 45,300원   │← 상단 바
├─────────────────────────────────┤
│                                 │
│           [다방 일러스트]         │
│         (고양이 주인공)           │
│                                 │
│         "오늘의 차는?"            │
├─────────────────────────────────┤
│                                 │
│  [📞 손님 대기 중!]  (클릭)       │
│   다음 손님이 올 때까지...        │
│                                 │
├─────────────────────────────────┤
│ [시작] [상점] [컬렉션] [설정]    │← 하단 네비게이션
└─────────────────────────────────┘
```

### 2.2 요소 상세

| 요소 | 크기 | 색상 | 기능 |
|------|------|------|------|
| 상단 바 | 80px | #F5F0E6 (베이지) | 게임 정보 표시 |
| 다방 일러스트 | 400×300px | - | 게임 분위기 설정 |
| 손님 대기 버튼 | 280×80px | #FF9F43 (주황) | 손님 생성 트리거 |
| 하단 네비게이션 | 120px | #FFF8DC | 메뉴 이동 |

### 2.3 상태별 표시

- **손님 없음**: "손님 대기 중" (주황, 클릭 가능)
- **손님 대기**: "손님이 왔어요!" (빨강, 깜박임)
- **손님 서빙 중**: 주문 화면으로 자동 전환

## 3. 주문 화면 (Order Screen)

### 3.1 구성

```
┌──────────────────────────────────┐
│ [← 돌아가기]     [손님 이름 / 아이콘]│
├──────────────────────────────────┤
│        손님 대사:                  │
│    "따뜻한 유자차 주세요"          │
│    (말풍선)                        │
├──────────────────────────────────┤
│  요청사항:                         │
│  ☑️ 따뜻한 온도                    │
│  ☑️ 약 3분 우린 것                │
│                                  │
├──────────────────────────────────┤
│  차 선택:                          │
│  [유자차] [말차] [보이차]           │
│  [연꽃차] [차이]                   │
│  (선택 불가능한 차는 회색)         │
├──────────────────────────────────┤
│          [차 선택 완료] →          │
└──────────────────────────────────┘
```

### 3.2 요청사항 아이콘

| 아이콘 | 의미 | 표시 방식 |
|--------|------|---------|
| 🌡️ | 온도 요구 | "따뜻함" / "중간" / "시원함" |
| ⏱️ | 우림 시간 | "짧게" / "중간" / "길게" |
| 🌿 | 재료 추가 | "꿀이 들어간" / "우유가 들어간" |

### 3.3 터치 영역

- 차 선택 버튼: 각 120×120px, 간격 16px
- 완료 버튼: 280×60px

## 4. 제조 화면 (Brewing Screen)

### 4.1 인터랙션 기반 구성

```
┌──────────────────────────────────┐
│ 제조 진행: [████████░░] 80%       │
├──────────────────────────────────┤
│                                  │
│        [냄비 일러스트]             │
│      (애니메이션: 물 끓기)         │
│                                  │
│   "3분간 우리는 중..."             │
│   (카운트다운: 2:45)               │
│                                  │
├──────────────────────────────────┤
│  온도 조정:                        │
│  [← 낮음] [중간 ●] [높음 →]       │
│                                  │
│  우림 시간 조정:                   │
│  [← 짧게] [중간 ●] [길게 →]       │
│                                  │
├──────────────────────────────────┤
│  추가 재료:                        │
│  [꿀 추가?] [우유 추가?]           │
│  (각 300원 차감)                   │
├──────────────────────────────────┤
│        [완료 및 평가로]            │
└──────────────────────────────────┘
```

### 4.2 인터랙션

- **온도 슬라이더**: 드래그로 3단계 선택
- **시간 슬라이더**: 드래그로 3단계 선택
- **재료 버튼**: 탭해서 추가 (회색 = 이미 추가됨)
- **완료 버튼**: 타이머 끝나거나 수동 클릭

### 4.3 상태 표시

| 온도 | 표시 | 색상 |
|------|------|------|
| 낮음 | ❄️ | 파랑 |
| 중간 | 🌡️ | 주황 |
| 높음 | 🔥 | 빨강 |

```

│     "음, 이 맛은... 좋네!"         │
│      (손님 반응 대사)              │
│                                  │
│     ⭐⭐⭐⭐⭐                      │
│      5.0 점 만족!                 │
│                                  │
├──────────────────────────────────┤
│  보상:                            │
│  💰 획득 금액: 4,500원             │
│  ✨ 획득 경험치: 600 EXP           │
│  💝 팁: 750원 추가!                │
│                                  │
│  총 수익: 5,250원                 │
│                                  │
├──────────────────────────────────┤
```

| 3⭐ | 3개 채움 | 보통 | 1.0배 |
| 2⭐ | 2개 채움 | 아쉬움 | 0.7배 |
| 1⭐ | 1개 채움 | 불만족 | 0.3배 |

## 6. 상점 화면 (Shop Screen)

### 6.1 탭 구조

```
┌──────────────────────────────────┐
│  [재료] | [인테리어] | [문화카드]  │← 탭
├──────────────────────────────────┤
│                                  │
│  재료 선택:                        │
│  [꿀 300원] [우유 400원]           │
│  [생강 200원] [계피 250원]         │
│  [레몬 150원] [생크림 600원]       │
│  [정향 200원] [카다몬 300원]       │
│  [얼음 100원]                     │
│                                  │
│  선택된 재료:                      │
│  총 비용: 1,200원                  │
│  [구매 확인]                       │
│                                  │
└──────────────────────────────────┘
```

### 6.2 탭별 내용

**[재료]**
- 9가지 재료 표시
- 이미 구매한 재료: 체크마크 표시
- 개별 구매 또는 다중 선택 가능

**[인테리어]**
- 벽지, 조명, 테이블, 장식품
- 레벨별 해금 표시 (🔒 Lv.5)
- 미리보기 이미지
- 구매한 아이템: 체크마크

**[문화카드]**
- 수집한 카드 표시
- 아직 못 본 카드: 물음표 표시
- 카드 클릭시 상세 정보 팝업
