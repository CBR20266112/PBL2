## 1. 주인공 - 고양이 (Protagonist Cat)

### 1.1 기본 설정

| 항목 | 설명 |
|------|------|
| **이름** | orangeCat (임시) → 게임 시작 시 플레이어 지정 팝업 |
| **종족** | 고양이 |
| **성별** | 여성 (부드러움 추구) |
| **나이** | 20-25세 (추정) |
| **성격** | 부드럽고, 치열하고, 배우는 것을 좋아함 |
| **직업** | 전통 차 카페 주인 (할머니로부터 상속) |

### 1.2 시각적 특징

**외형**
- 색상: 흰색 + 회색 (양쪽 귀 포인트)
- 크기: 일반 고양이 크기 (비현실적이지만 귀엽게)
- 눈: 크고 동그란 형태, 갈색 또는 연초록
- 꼬리: 온화한 움직임

**복장** (기본)
- 앞치마: 베이지색 앞치마 (흰색 스트라이프)
- 상의: 흰색 긴팔 셔츠
- 하의: 회색 편한 바지

**특징 장신구**
- 팔찌: 할머니로부터 받은 옥 팔찌 (스토리 아이템)
- 앞치마 포켓: 손님 요청사항 메모 보관

### 1.3 감정 표현

| 감정 | 표현 방식 | 상황 |
|------|---------|------|
| 행복 | 눈 반달 모양, 입 웃음 | 손님 만족도 높음 |
| 집중 | 눈 크게 뜸, 입 다물기 | 제조 중 |
| 놀람 | 귀 쫑긋, 눈 놀람 표현 | 새로운 손님 방문 |
| 피곤 | 눈 반쯤 감기, 입 약간 벌림 | 연속 손님 서빙 후 |
| 슬픔 | 눈 젖어있음, 입 처짐 | 손님 만족도 낮음 |
| 생각중 | 손가락 턱에, 표정 진지 | 선택지 제시 시 |

### 1.4 애니메이션

**상시 애니메이션**
- 아이들 상태: 미세한 꼬리 흔들기 (2초 주기)
- 호흡: 가슴 미세 상하운동

**상황별 애니메이션**
- 손님 맞이: 인사 손 흔들기 (1초)
- 제조 중: 팔 움직임으로 냄비 저으며 표현
- 보상 획득: 점프 애니메이션
- 스트레스: 귀 아래로 처지는 표현

## 2. 손님 1 - Luna (루나, 토끼 여성)

### 2.1 프로필

| 항목 | 설명 |
|------|------|
| **이름** | Luna (루나) |
| **종족** | 토끼 |
| **나이** | 18-20세 (어린 학생) |
| **성격** | 수줍음, 깜깜한 것 좋아함, 예민함 |
| **직업** | 미술대학 학생 |
| **선호 차** | 유자차 (편안함) |
| **주요 특징** | 밤 늦게 방문 (과제 때문) |

### 2.2 시각적 특징

**외형**
- 색상: 흰색 + 핑크색 코와 귀 안쪽
- 눈: 크고 맑은 보라색
- 귀: 길고 솰솰한 느낌
- 크기: 작고 연약해 보임

**복장**
- 후드 덮개가 있는 어두운 까운
- 스니커 또는 슬리퍼
- 가방: 미술 도구 가방

**표정**
- 기본: 약간 불안한 표정
- 만족: 조용한 미소, 눈 빛남
- 불만족: 눈 돌림, 고개 외로움

### 2.3 친숙도별 스토리

**Lv.1-3 (낯선 사람)**
- 항상 혼자 옴
- 유자차만 주문
- 말을 적게 함

**Lv.4-7 (친한 손님)**
- 가끔 친구와 옴
- 새로운 차 시도해봄
- 과제 이야기하기 시작

**Lv.8-10 (가족처럼)**
- 항상 유자차 + 추가 재료 요청
- 다링에게 그림 선물
- 밤새우는 습관 걱정 표현

## 3. 손님 2 - Hyuntae (현태, 사자 남성)

### 3.1 프로필

| 항목 | 설명 |
|------|------|
| **이름** | Hyuntae (현태) |
| **종족** | 사자 |
| **나이** | 30-35세 (어른) |
| **성격** | 활발함, 따뜻함, 리더십 있음 |
| **직업** | 카페 체인 CEO (잠재적 비즈니스 파트너) |
| **선호 차** | 차이 (강한 맛) |
| **주요 특징** | 낮 12시-1시 방문 (회의 사이) |

### 3.2 시각적 특징

**외형**
- 색상: 황금색 갈기, 황갈색 몸
- 눈: 진한 갈색, 자신감 넘침
- 갈기: 풍성하고 정돈된 모습
- 크기: 크고 위엄 있음

**복장**
- 고급 정장 (회의 복장)
- 넥타이 또는 스카프
- 손목시계: 고급 시계

**표정**
- 기본: 자신 있는 미소
- 만족: 활짝 웃음, 엄지손가락
- 불만족: 찌푸린 표정

### 3.3 친숙도별 스토리

**Lv.1-3 (낯선 사람)**
- 바쁜 척함
- 항상 서 있음
- 시간에 쫓김

**Lv.4-7 (친한 손님)**
- 앉아서 즐기기 시작
- 차 맛에 진지해짐
- 다링의 사업 센스 칭찬

**Lv.8-10 (가족처럼)**
- 협력 제안
- 차 공급 계약 논의
- 비즈니스 멘토 역할 시작

## 4. 손님 3 - Wei (웨이, 판다 남성)

### 4.1 프로필

| 항목 | 설명 |
|------|------|
| **이름** | Wei (웨이) |
| **종족** | 판다 |
| **나이** | 25-28세 |
| **성격** | 차분함, 과학적 사고, 고급스러움 |
| **직업** | 건강식 영양사 |
| **선호 차** | 보이차 (건강식) |
| **주요 특징** | 주말에 방문 (쇼핑 사이) |

### 4.2 시각적 특징

**외형**
- 색상: 검은색 + 흰색 (판다의 전형적)
- 눈: 둥글고 따뜻함
- 체형: 통통하고 친근해 보임
- 크기: 중간~큼

**복장**
- 캐주얼 청바지
- 면 셔츠 또는 니트
- 백팩: 건강식품 담음

**표정**
- 기본: 미소 띤 평온함
- 만족: 만족스러운 표정
- 불만족: 생각 깊어짐

### 4.3 친숙도별 스토리

**Lv.1-3 (낯선 사람)**
- 혼자 옴
- 보이차만 주문
- 건강 정보 확인함

**Lv.4-7 (친한 손님)**
- 다링에게 건강 조언
- 다양한 차 효능 질문
- 건강 이야기로 수다

**Lv.8-10 (가족처럼)**
- 차의 영양학적 가치 함께 논의
- 특별 건강 음료 제안
- 다링의 건강 염려

## 5. 손님 4 - Sakura (사쿠라, 여우 여성)

### 5.1 프로필

| 항목 | 설명 |
|------|------|
| **이름** | Sakura (사쿠라) |
| **종족** | 여우 |
| **나이** | 22-25세 |
| **성격** | 활기찬, 사교적, 트렌디함 |
| **직업** | SNS 크리에이터 (영향력 있음) |
| **선호 차** | 말차 (예쁜 외모) |
| **주요 특징** | 오후 3-4시 방문 (사진 촬영) |

### 5.2 시각적 특징

**외형**
- 색상: 주황색 + 흰색
- 눈: 똑똑하고 반짝임
- 꼬리: 풍성하고 우아함
- 크기: 중간~작음

**복장**
- 유행하는 옷 (시즌별 변경)
- 악세서리 많음
- 카메라: 항상 소지

**표정**
- 기본: 카리스마 있는 미소
- 만족: 신나는 표정
- 불만족: 실망한 표정

### 5.3 친숙도별 스토리

**Lv.1-3 (낯선 사람)**
- 야 사진만 찍음
- 말이 많음
- 빨리 떠남

**Lv.4-7 (친한 손님)**
- 다링과 함께 사진 찍음
- 소셜 미디어 홍보 시작
- 다방 유명해짐

**Lv.8-10 (가족처럼)**
- SNS 명소 되기
- 콜라보레이션 제안
- 다링의 팬 되기

## 6. 손님 5 - Denu (데누, 독수리 남성)

### 6.1 프로필

| 항목 | 설명 |
|------|------|
| **이름** | Denu (데누) |
| **종족** | 독수리 |
| **나이** | 60세 초반 (어르신) |
| **성격** | 침착함, 지혜로움, 신비로움 |
| **직업** | 은퇴한 차 수집가 |
| **선호 차** | 차이 (깊은 맛) |
| **주요 특징** | 불규칙하게 방문 (여행 많음) |

### 6.2 시각적 특징

**외형**
- 색상: 검은색 + 흰색 (독수리의 전형)
- 눈: 크고 통찰력 있음
- 부리: 우아하고 정돈됨
- 크기: 크고 위엄 있음

**복장**
- 전통 한복 또는 중국 스타일 복장
- 목주머니: 팔찌 하나
- 지팡이 (선택적)

**표정**
- 기본: 신비로운 미소
- 만족: 흐뭇한 표정
- 불만족: 걱정스러운 표정

### 6.3 친숙도별 스토리 (스토리 핵심)

**Lv.1-3 (낯선 사람)**
- 첫 방문이 언제인지 불명확
- 다링을 관찰함
- 할머니에 대해 물음

**Lv.4-7 (친한 손님)**
- 할머니와의 옛날 이야기 공유
- 차의 깊이를 다링과 함께 탐구
- 신비로운 조언 주기

**Lv.8-10 (가족처럼)** ⭐ 스토리 전환점
- 다링의 할머니와 옛날 친구였음 밝혀짐
- 할머니 안부 묻기
- 다링에게 중요한 유산 전달

## 7. 감정 애니메이션 가이드

### 모든 캐릭터 공통

| 감정 | 눈 표현 | 입 표현 | 신체 | 길이 |
|------|--------|--------|------|------|
| 행복 | 반달 모양 | 웃음 | 앞으로 기울기 | 0.5초 |
| 놀람 | 크게 뜨기 | O자 입 | 뒤로 물러나기 | 0.3초 |

| 분노 | 가늘게 | 앙다물기 | 발 구르기 | 1초 |
| 생각 | 정면 | 턱 괴기 | 한 손 턱 | 1.5초 |

## 8. 콘셉트 아트 참고

### 시각적 일관성 원칙

1. **모두 동물의 의인화**: 사람처럼 옷 입고 행동하되 동물 특징은 유지
2. **둥근 형태**: 모든 모서리를 둥글게 (귀엽고 따뜻함)
3. **색상 조화**: 각 캐릭터 고유 색상 팔레트 (섞이지 않음)
4. **표정 과장**: 감정 명확히 표현 (사람처럼이 아닌 캐릭터처럼)
5. **크기 다양성**: 종족별로 크기 차이 표현

## 10. 특별 상황별 표현

### 특별 날씨

- **비 오는 날**: 모든 캐릭터 축축해 보임, 옷 더 어두움
- **눈 오는 날**: 할머니 추억 표현 (Denu는 향수 표정)
- **봄**: 벚꽃 테마 (Sakura는 꽃 테마 옷)

### 계절 복장

| 계절 | 변화 |
|------|------|
| 봄 | 가벼운 옷 |
| 여름 | 짧은 소매, 밝은 색 |
| 가을 | 따뜻한 색 |
| 겨울 | 스카프, 목도리 |

## 11. 캐릭터 성장 표현

### 친숙도 높아질 때

- 처음: 경직된 자세 → 편안한 자세
- 눈: 경계하는 눈 → 믿음의 눈
- 표정: 어색함 → 자연스러움
- 대사: 짧음 → 길고 개인적
