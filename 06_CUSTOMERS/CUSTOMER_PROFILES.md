# 손님 상세 프로필 (Customer Profiles)

**버전**: v0.2.0  
**마지막 수정**: 2026-07-11

---

## 1. 손님 시스템 개요

### 1.1 손님의 역할
- 게임의 주된 상호작용 대상
- 플레이어의 성공/실패 평가자
- 스토리 진행의 중심축
- 감정 입입 및 플레이어 보상 동기 제공

### 1.2 손님 방문 시스템
- **영업시간**: 11:00 ~ 18:00
- **일일 손님**: 3-5명 (랜덤)
- **특정 손님**: 특정 시간대에 주로 방문
- **친숙도**: 자주 만날수록 높아짐

## 2. 손님 1: 루나 (Luna) - 귀여운 토끼

### 기본 정보
| 항목 | 내용 |
|------|------|
| 이름 | 루나 (Luna) |
| 종 | 토끼 |
| 색상 | 흰색 + 분홍색 귀 |
| 나이 | 20대 초반 |
| 성격 | 밝고 수다스러움 |
| 직업 | 대학생 |
| 선호 시간대 | 오후 2-3시 |

### 선호도
| 차 이름 | 선호도 | 이유 |
|--------|--------|------|
| 유자차 | ⭐⭐⭐⭐⭐ | 상큼한 맛을 가장 좋아함 |
| 말차 | ⭐⭐⭐ | 좋아하지만 쓸 수 있음 |
| 보이차 | ⭐⭐ | 너무 진함 |
| 연꽃차 | ⭐⭐⭐⭐ | 예뻐서 좋아함 |
| 차이 | ⭐ | 너무 따뜻함 |

### 주문 패턴
- **기본 주문**: 유자차 + 꿀 (가장 자주)
- **기분 좋을 때**: 연꽃차
- **스트레스 받을 때**: 상큼한 음료 원함
- **평균 구매액**: 4,000-5,500원

### 개인 스토리
루나는 대학생으로 매일 도서관에 가기 전에 들린다.
시험 기간에는 더 자주 방문하며, 스트레스를 많이 받는다.
플레이어가 루나를 계속 만족시키면, 친구들을 소개하겠다고 말한다.

### 특수 상호작용
- **특별 대사**: "이번주 수학시험이 정말 힘들어요..."
- **좋아하는 반응**: "우아, 이건 정말 맛있다!"
- **싫어하는 반응**: "어? 이건 좀 진한데..."
- **친숙도 최대**: "너희 다방이 제일 좋아!"

### 스토리 진행
**Lv.1-5**: 루나 첫 만남, 기본 대사만
**Lv.6-10**: 루나가 개인 고민 얘기 시작
**Lv.11+**: 루나의 가족 배경 공개, 깊은 우정 형성

### 선호도

|--------|--------|------|

게임 진행 중 현태가 같은 동물 여자친구를 데려오게 된다.

- **싫어하는 반응**: "이건 제 입맛이 아니네요."
- **친숙도 최대**: "여기가 나의 작은 휴식처입니다."

### 스토리 진행
**Lv.1-8**: 현태의 외로움 드러남
**Lv.9-15**: 현태의 로맨스 서브스토리
**Lv.16+**: 현태가 다방에서 청혼하고 싶어함

## 7. 친숙도 시스템

### 친숙도 단계
| 단계 | 대사 | 보상 |
|------|------|------|
| 낯선 사람 | 짧은 인사만 | 기본 팁 |
| 알고 지내는 사이 | 일상 얘기 | 팁 10% 증가 |
| 친구 | 개인 고민 공유 | 팁 20% 증가 |
| 절친 | 깊은 감정 나눔 | 팁 30% 증가 + 특별 선물 |
| 가족 같은 사이 | 스토리 완성 | 팁 50% 증가 + 스토리 엔딩 |

### 친숙도 상승 조건
- 요청한 차를 정확히 제공
- 높은 만족도 (별 5개)
- 지속적인 방문 (계속 만남)
- 특별한 요청 만족 (추후 추가)

## 9. 손님 상호작용 시스템

### 주문 표시
손님이 들어올 때 원하는 차를 시각적으로 표시:
- 말풍선에 차 이름 표시
- 차의 색상으로 표시 (색맹도 고려)
- 텍스트 + 아이콘 병행

### 제공 실패 시 반응
- 틀린 차 제공: "어? 이건 내가 주문한 게 아닌데..."
- 온도 부적절: "어? 뜨거워/차가워..."
- 우림시간 부적절: "어? 이건 너무 연해/진한데..."

### 성공 시 반응 (별점)
- **5별**: "최고야! 정말 감사해!"
- **4별**: "맛있어! 고마워!"
- **3별**: "괜찮은데... 좀 아쉽네."
- **2별**: "이건... 좀 이상한데?"
- **1별**: "이건 아니야..."
